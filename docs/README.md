Esta é uma versão de demonstração estática do Nexus Intranet hospedada no GitHub Pages.

- [Dashboard](index.html)

## ⚠️ Aviso

Esta é uma **versão estática de demonstração**. O sistema completo com funcionalidades dinâmicas requer:

- PHP 8.2+
- MySQL 8.0+
- Servidor web (Apache/Nginx)

## 🚀 Funcionalidades do Sistema Completo

- ✅ Dashboard com métricas em tempo real
- ✅ Quadro Kanban com drag-and-drop
- ✅ Gestão de contratos com alertas de vencimento
- ✅ Diretório de sites e links rápidos
- ✅ Chat corporativo com assistente IA
- ✅ Sistema de categorias/pastas
- ✅ Logs de auditoria
- ✅ Interface responsiva com Tailwind CSS
